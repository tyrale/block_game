# tetris
A minimal Tetris game for macOS, in a Swift package. A port of the [Tetris example in the Vlang repository](https://github.com/vlang/v/tree/master/examples/tetris).

<picture>
  <img src="tetris.gif" width="240" height="500">
</picture>

### Build

```swift
swift build
```

### Run
```swift
swift run
```
